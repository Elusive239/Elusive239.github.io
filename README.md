# Elusive239.github.io
A website about me (Eventually!)! First I wanted to write some kind of basic markdown parser...

[Markdown Spec](https://commonmark.org/help/)

### Supported Syntax

| Key | Meaning |
| :-: | :-|
| [ ] | Not Started|
| [-] | In Progress|
| [X] | Supported!|

| Supported | Syntax |
| :---------------: | :------- |
| [X] | * *Italic* |
| [X] | _ *Italic* |
| [X] | ** **Bold** |
| [X] | __ **Bold** |
| [X] | # Headings|
| [ ] | ==========|
| [ ] | ----------|
| [X] | [Links](...)|
| [X] | ![Image](...)|
| [X] | [Links][...]|
| [X] | ![Image][...]|
| [X] | > Blockquotes|
| [X] | * List|
| [X] | - List|
| [X] | 1. List|
| [X] | 1) List|
| [X] | --- Horizontal Rule|
| [X] | *** Horizontal Rule|
| [X] | `Inline Code`|
| [X] | ``` \`\`\`Code Block\`\`\` ```|
| [ ] | ``` ....Code Block ```|
